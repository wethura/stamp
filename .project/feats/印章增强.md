# 印章增强

StampData 拆分为 StampTemplate（纯模板）+ StampInstance（页级配置），实现同一模板在同一文档的多实例、多页独立配置。

## 功能点

- **模板+实例分离**：StampTemplate 只含元数据（id/name/image），全局持久化；StampInstance 含页级配置（位置/大小/透明度/旋转），纯内存管理
- **模板库交互**：章列表作为模板库，双击模板或拖拽到预览区在当前页创建实例
- **每页独立配置**：同一模板可在不同页创建独立实例，每实例有独立的 pos_x/pos_y/size_ratio/opacity/rotation
- **预览区交互**：显示当前页所有实例，点击选中进入编辑状态，拖拽调整位置，Backspace/右键菜单删除
- **旋转支持**：0°~360° 任意角度，Pillow `Image.rotate(expand=True)` 预旋转后插入
- **纵横页面适配**：印章大小基于每页实际宽度计算，混合方向文档自动适配
- **多实例叠加导出**：每页按实例顺序逐个叠加，支持 PDF/图片/Excel
